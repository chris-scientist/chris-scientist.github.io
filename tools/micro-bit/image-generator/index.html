Goal of this tool

This tool generate C++ array of boolean, which represents a static image for my micro:bit API.

Help

Toolbar

  • Reverse : switch the state of all LEDs, if the state is true then to set at false, else if the state is false then to set at true.
  • Turn ON all LEDs : all state LEDs are true when you click on this button.
  • Turn OFF all LEDs : all state LEDs are false when you click on this button.

Workflow

  1. Design your static image by clicking on the LEDs of the matrix.
  2. Copy the source code in your C++ application.
  3. That's all !
